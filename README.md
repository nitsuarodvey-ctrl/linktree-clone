# Alphi Links 🔗

[TR] Alphi Links, sosyal medya hesaplarınızı, portfolyonuzu ve önemli bağlantılarınızı tek bir şık çatıda toplayan, Glassmorphism tasarım diline sahip modern bir "link-in-bio" aracıdır.
[EN] Alphi Links is a modern "link-in-bio" tool featuring Glassmorphism design, allowing you to gather your social media, portfolio, and essential links under one stylish page.

---

## ✨ Özellikler / Features

### 🇹🇷 Türkçe
- **Canlı Önizleme:** Sol taraftaki editörde yaptığınız değişiklikleri sağdaki mobil simülasyonda anlık olarak görün.
- **Glassmorphism Arayüzü:** Backdrop-filter ve modern cam efektleri ile estetik tasarım.
- **Dinamik Profil Yönetimi:** Profil fotoğrafı yükleme, isim ve konum bilgilerini anlık güncelleme.
- **Sınırsız Link Ekleme:** Dilediğiniz kadar bağlantı ekleyin, düzenleyin veya silin.

### 🇺🇸 English
- **Real-time Preview:** See your edits instantly on the mobile simulation as you type in the editor.
- **Glassmorphism UI:** Sleek aesthetic design using modern backdrop filters and glass effects.
- **Dynamic Profile Management:** Instant updates for profile pictures, names, and location data.
- **Unlimited Links:** Add, edit, or remove as many links as you need dynamically.

---

## 🛠️ Teknik Yığın / Tech Stack

- **Frontend:** HTML5, Tailwind CSS
- **Design System:** Glassmorphism & Mobile-First Preview
- **Logic:** Vanilla JavaScript (FileReader API & DOM Manipulation)
- **Typography:** Inter via Google Fonts

---

## 🚀 Kullanım / How to Use

[TR] Kurulum gerektirmez. `index.html` dosyasını tarayıcıda açarak kendi link sayfanızı oluşturmaya başlayabilirsiniz.
[EN] No installation required. Simply open `index.html` in your browser to start building your personal link page.
